Netflix Data Analysis

Dataset

Netflix Titles Dataset from Kaggle.

Problem Statements

1. Movies vs TV Shows
2. Most Common Ratings
3. Content Growth Over Time
4. Top Countries
5. Most Common Genres

Tools Used

* Python
* pandas
* matplotlib
* seaborn
* Jupyter Notebook

Key Findings

* Movies dominate the catalog.
* TV-MA is the most frequent rating.
* Netflix content increased rapidly after 2015.
* United States contributes the most content.
* Drama-related genres are highly common.
